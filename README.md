# 🔐 Login Page UI

This project is a modern and visually appealing login page designed using HTML and CSS. It follows the glassmorphism design trend with a blurred background, soft shadows, and clean user interface.

## ✨ Features

- Glassmorphism UI design
- Blurred background effect
- Responsive layout
- Font Awesome icons integration
- Smooth hover and focus effects
- Clean and minimal design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome

## 📷 Preview

A stylish login form with username and password fields, remember me option, and register link.
